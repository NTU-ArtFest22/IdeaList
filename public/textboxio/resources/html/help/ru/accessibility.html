Специальные возможности
Включение управления с помощью клавиатуры

Для включения управления панелью инструментов с помощью клавиатуры нажмите клавишу F10 (Windows) или сочетание клавиш ALT + F10 (Mac OSX). Первый элемент на панели инструментов будет выделен синей рамкой. Это значит, что он выбран.

Перемещение между группами

Кнопки на панели инструментов разделены на группы похожих действий. При включенном управлении с помощью клавиатуры нажатие клавиши Tab осуществляет выбор следующей группы, а нажатие сочетания клавиш Shift + Tab — предыдущей группы. Если нажать клавишу Tab, когда выбрана последняя группа, будет выбрана первая группа кнопок.

Перемещение между элементами или кнопками

Для перемещения между элементами используйте клавиши со стрелками. В пределах групп выбор элементов осуществляется по кругу. Для перехода к следующей группе нажмите клавишу Tab. Для перемещения в пределах группы используйте клавиши со стрелками.

Выполнение команд

Для выполнения команды выберите нужную кнопку и нажмите клавишу пробела или Enter.

Открытие, закрытие меню и перемещение по нему

Если кнопка на панели инструментов содержит меню, выберите ее и нажмите клавишу пробела или Enter. На экране отобразится меню, в котором будет выбран первый пункт. Для перемещения по меню используйте клавиши со стрелками. Для перемещения вверх и вниз по меню и подменю используйте клавиши со стрелками «вверх» и «вниз» соответственно.

Если пункт меню располагает подменю, этот пункт будет отмечен символом угловых скобок. Для вызова подменю можно использовать клавишу со стрелкой, направление которой соответствует направлению символа угловых скобок. Нажатие клавиши со стрелкой в обратном направлении приведет к закрытию подменю.

Чтобы закрыть любое активное меню, нажмите клавишу Esc. После закрытия меню снова будет выбран предыдущий элемент.

Редактирование или удаление гиперссылок

Для редактирования или удаления ссылки вызовите меню «Вставка» и выберите пункт «Вставка ссылки». На экране редактора появится диалоговое окно редактирования ссылок.

Для редактирования ссылки введите новый URL-адрес в текстовое поле обновления ссылки, затем нажмите клавишу Enter. Для удаления ссылки из документа нажмите кнопку «Удалить». Для выхода из окна без сохранения изменений нажмите клавишу Esc.

Изменение размера шрифта и границы таблицы

Для изменения размера шрифта вызовите меню шрифтов и выберите размер. На экране редактора появится поле изменения размера, которое станет активным.

Для изменения размера границы выберите на панели инструментов элемент редактирования размеров границ таблицы и выберите размер. На экране редактора появится поле изменения размера, которое станет активным. Примечание. Расположенный на панели инструментов элемент изменения размеров границ таблицы активен, только когда курсор находится внутри таблицы.

Находясь в диалоговом окне, нажмите клавишу Tab для выбора следующего элемента управления. Для выбора предыдущего элемента управления нажмите сочетание клавиш Shift + Tab.

Измените размер, введя новое значение в поле размера. Например, 14px или 1em. Чтобы принять изменения, нажмите клавишу Enter. Обратите внимание, что нажатие этой клавиши приведет к закрытию диалогового окна. При этом станет активным окно документа.

Для изменения размера без выхода из диалогового окна используйте кнопки увеличения и уменьшения размера. Изменение размера с помощью этих кнопок приведет к мгновенному изменению размера выбранного элемента с сохранением текущего значения. Для выхода из диалогового окна изменения размеров нажмите клавишу Esc.
