Instruções de acessibilidade
Ativar a navegação pelo teclado

Para ativar a navegação pelo teclado na barra de ferramentas, prima F10 para Windows ou ALT + F10 no Mac OSX. O primeiro item na barra de ferramentas será realçado com um contorno azul, indicando que está selecionado.

Movimentar-se entre grupos

Os botões na barra de ferramentas com funções assemelhadas são separados em grupos. Quando a navegação pelo teclado estiver ativa, prima a tecla de tabulação para mover a seleção para o próximo grupo e Shift+Tab para mover a seleção para o grupo anterior. Ao premir a tecla de tabulação no último grupo, retorna ao primeiro grupo de botões.

Movimentar-se entre itens ou botões

Para voltar e avançar entre os itens, use as teclas de seta. Os itens serão selecionados em sequência dentro do próprio grupo. Para ir para o grupo seguinte, prima a tecla de tabulação e use as teclas de seta.

Executar comandos

Para executar um comando, leve a seleção até ao botão desejado e prima espaço ou Enter.

Abrir, navegar e fechar menus

Quando um botão da barra de ferramentas contiver um menu, prima a barra de espaços ou Enter para abrir o menu. Quando o menu abrir, o primeiro item estará selecionado. Use as teclas de seta para navegar pelo menu. Para subir ou descer pelo menu, prima as teclas de seta para cima e para baixo, respetivamente. Isto também se aplica aos submenus.

Os itens de menu que possuem submenus são identificados por um símbolo de divisa. Use a tecla de seta que corresponde à direção da divisa para expandir o submenu, e a tecla de seta oposta para fechá-lo.

Para fechar qualquer menu ativo, prima a tecla Esc. Quando um menu é fechado, a seleção volta à posição anterior.

Modificar e remover hiperligações

Para editar ou remover uma hiperligação, navegue até ao menu Inserir e selecione Inserir hiperligação. O editor exibe a caixa de diálogo de edição de hiperligação.

Edite a hiperligação inserindo o novo url na caixa de entrada de atualização de hiperligação e premindo Enter. Remova a hiperligação do documento selecionando o botão de remover. Para sair da caixa de diálogo sem fazer alterações, prima Esc.

Alterar o tamanho do tipo de letra e do limite da tabela

Altere o tamanho do tipo de letra navegando até ao menu de tipo de letra e selecionando o tamanho. O editor exibe a caixa de diálogo de tamanho no menu e define o foco nela.

Altere o tamanho do limite da tabela navegando até ao item relacionado da barra de ferramentas e selecionando o tamanho do limite. O editor exibe a caixa de diálogo de tamanho no menu e define o foco nela. Nota: o item da barra de ferramentas para dimensionar o limite da tabela só estará disponível quando o cursor estiver dentro de uma tabela.

Na caixa de diálogo de tamanho, prima a tecla de tabulação para mover a seleção para o controlo seguinte. Prima Shift+Tab para mover a seleção para o controlo anterior.

Modifique o tamanho inserindo o novo valor na caixa de entrada de tamanho. Por exemplo, 14px ou 1em. Para submeter as alterações, prima Enter. Observe que, ao premir Enter, fecha a caixa de diálogo e volta o foco ao documento.

Altere o tamanho sem sair da caixa de diálogo ativando os botões de aumentar tamanho ou diminuir tamanho. A alteração de tamanho com os botões de aumentar ou diminuir modificará imediatamente o tamanho do elemento selecionado, enquanto o valor de unidade em uso é mantido. Saia da caixa de diálogo premindo Esc.
