Μορφοποίηση Markdown

Η Markdown είναι μια σύνταξη για τη δημιουργία δομών HTML και τη μορφοποίηση χωρίς να απαιτείται η χρήση συντομεύσεων πληκτρολογίου ή μενού πρόσβασης. Για να χρησιμοποιήσετε τη μορφοποίηση markdown, εισαγάγετε την επιθυμητή σύνταξη ακολουθούμενη από το πλήκτρο διαστήματος ή enter.
