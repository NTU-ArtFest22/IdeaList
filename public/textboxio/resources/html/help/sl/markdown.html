Oblikovanje Markdown

Markdown je sintaksa za ustvarjanje struktur HTML in oblikovanje brez uporabe bližnjic na tipkovnici ali menijev dostopa. Če želite uporabiti oblikovanje Markdown, vnesite želeno sintakso in pritisnite Enter ali tipko za presledek.

Sintaksa oblikovanja s tipkovnico
Sintaksa Rezultat HTML 
# Največji naslov
 
<h1> Največji naslov</h1>
 
## Večji naslov
 
<h2>Večji naslov</h2>
 
### Veliki naslov
 
<h3>Veliki naslov</h3>
 
####  Naslov
 
<h4>Naslov</h4>
 
##### Majhen naslov
 
<h5>Majhen naslov</h5>
 
###### Najmanjši naslov
 
<h6>Najmanjši naslov</h6>
 
Neurejen seznam
 
<ul><li>Neurejen seznam</li></ul>
