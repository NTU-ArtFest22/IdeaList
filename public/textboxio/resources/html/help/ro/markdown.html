Formatare Markdown

Markdown este o sintaxă pentru crearea structurilor HTML şi formatare fără utilizarea comenzilor rapide de la tastatură sau accesarea meniurilor. Pentru a utiliza formatarea markdown, introduceţi sintaxa dorită urmată de apăsarea tastei enter sau space.

Sintaxă de formatare tastatură
Sintaxă Rezultat HTML 
# Cel mai mare antet
 
<h1> Cel mai mare antet</h1>
 
## Antet mai mare
 
<h2>Antet mai mare</h2>
 
### Antet mare
 
<h3>Antet mare</h3>
 
####  Antet
 
<h4>Antet</h4>
 
##### Antet mic
 
<h5>Antet mic</h5>
 
###### Cel mai mic antet
 
<h6>Cel mai mic antet</h6>
