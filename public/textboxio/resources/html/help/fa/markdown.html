قالب بندی Markdown

Markdown یک ساختار دستوری برای ایجاد ساختارهای HTML و قالب بندی بدون نیاز به استفاده از میانبرهای صفحه کلید و دسترسی به منوها است. برای استفاده از قالب بندی markdown، ساختار دستوری دلخواه را با کلید enter یا space وارد کنید.
