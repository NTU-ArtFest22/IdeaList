Комбінації клавіш
Команди для роботи з редактором
Дія Windows Mac OS
Скасувати CTRL + Z ⌘Z
Повторити CTRL + Y ⌘⇧Z
Жирний CTRL + B ⌘B
Курсив CTRL + I ⌘I
Підкреслення CTRL + U ⌘U
Відступ CTRL + ] ⌘]
Зменшити відступ CTRL + [ ⌘[
Додати посилання CTRL + K ⌘K
Знайти CTRL + F ⌘F
Режим повного екрана (переключення) CTRL + SHIFT + F ⌘⇧F
Діалогове вікно «Допомога» (відкрити) CTRL + SHIFT + H ⌃⌥H
Контекстне меню (відкрити) SHIFT + F10 ⇧F10‎‏
Автоматичне заповнення коду CTRL + ПРОБІЛ ⌃ПРОБІЛ
*Примітка. Деякі функції можуть бути відключені вашим адміністратором.
