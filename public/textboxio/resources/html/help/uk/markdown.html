Форматування Markdown

Markdown – це синтаксична конструкція розмітки тексту для створення структур HTML і форматування без необхідності використання комбінацій клавіш та меню доступу. Для використання форматування Markdown необхідно ввести відповідну синтаксичну конструкцію та натиснути після цього Enter або пробіл.

Введення синтаксичної конструкції форматування за допомогою клавіатури
Синтаксична конструкція Результат в HTML
