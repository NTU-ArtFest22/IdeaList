Markdown-Formatierung

Markdown ist eine Syntax zum Erstellen von HTML-Strukturen und Formatierungen, ohne Tastenkombinationen verwenden oder auf Menüs zugreifen zu müssen. Geben Sie für die Markdown-Formatierung die gewünschte Syntax ein und drücken Sie dann auf die Eingabe- oder Leertaste.
