ชอร์ตคัตจากแป้นพิมพ์
คำสั่งในการปรับปรุงแก้ไข
แอ๊คชั่น Windows Mac OS
เปลี่ยนกลับคืน CTRL + Z ⌘Z
ทำใหม่อีก CTRL + Y ⌘⇧Z
ตัวหนา CTRL + B ⌘B
ตัวเอียง CTRL + I ⌘I
ขีดเส้นใต้ CTRL + U ⌘U
เยื้องย่อหน้า CTRL + ] ⌘]
ลดการเยื้องย่อหน้า CTRL + [ ⌘[
Add Link CTRL + K ⌘K
ค้นหา CTRL + F ⌘F
โหมดเต็มหน้าจอ (สลับไปมา) CTRL + SHIFT + F ⌘⇧F
กรอบสนทนาช่วยเหลือ (เปิด) CTRL + SHIFT + H ⌃⌥H
เมนูเนื้อหา (เปิด) SHIFT + F10 ⇧F10‎‏
ต่อคำอัติโนมัติสำหรับโค๊ต CTRL + SPACE ⌃Space
*หมายเหตุ: แอดมินสามารถปิดลักษณะบางอย่างได้
