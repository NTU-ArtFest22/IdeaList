Nedskrivningsformatering

Nedskrivningsformatering är syntax för att skapa HTML-strukturer och formatering utan att tangentbordsgenvägar eller åtkomstmenyer behöver användas. Ange önskad syntax följt av enter eller mellanslag för att använda nedskrivningsformatering.
