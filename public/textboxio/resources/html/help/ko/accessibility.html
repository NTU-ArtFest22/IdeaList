액세스 명령
키보드 탐색 활성화

도구 모음에서 키보드 탐색을 활성화하려면 Windows에서는 F10, Mac OSX에서는 ALT + F10을 누르십시오. 도구 모음의 첫 번째 항목이 선택되고 그 항목이 파란색 윤곽선으로 강조될 것입니다.

그룹 간 이동

도구 모음 내의 단추들은 비슷한 작업 그룹으로 구분되어 있습니다. 키보드 탐색이 활성화된 상태에서 tab 키를 누르면 다음 그룹이 선택되고, Shift + tab 키를 누르면 이전 그룹이 선택됩니다. 마지막 그룹이 선택된 상태에서 tab 키를 누르면 맨 첫 번째 단추 그룹이 다시 선택됩니다.

항목 또는 단추 간 이동

앞뒤로 항목 간 이동을 하려면 화살표 키를 사용하십시오. 항목은 해당 그룹 내에서만 이동합니다. 다음 그룹으로 건너 뛰려면 tab 키를 누르고 화살표 키를 사용하십시오.

명령 실행

명령을 실행하려면 원하는 해당 단추를 선택한 후 Space 키나 Enter 키를 누르십시오.

메뉴 열기, 탐색하기, 닫기

도구 모음 단추에 메뉴가 들어 있을 때 Space나 Enter를 누르면 메뉴가 열립니다. 메뉴가 열리면서 첫 번째 항목이 선택됩니다. 메뉴를 탐색하려면 화살표 키를 사용하십시오. 메뉴 위아래로 이동하려면 위쪽 또는 아래쪽 화살표 키를 각각 누르십시오. 이것은 하위 메뉴에서도 마찬가지입니다.

하위 메뉴가 있는 메뉴 항목은 펼침 단추 기호로 표시됩니다. 펼침 단추 방향의 화살표 키를 사용하면 하위 메뉴가 확장되고 펼침 반대 방향의 화살표 키를 사용하면 하위 메뉴가 닫힙니다.

활성 메뉴를 닫으려면 esc 키를 누르십시오. 메뉴가 닫히면 이전 선택으로 되돌아갑니다.

하이퍼링크 편집 또는 제거

링크를 편집하거나 제거하려면 삽입 메뉴로 가서 링크 삽입을 선택 하십시오. 편집기는 링크 편집 대화 상자를 표시합니다.

업데이트 링크 입력 상자에 새 url을 입력한 후 Enter 키를 눌러 링크를 편집하십시오. 문서에서 링크를 제거하려면 제거 단추를 선택하십시오. 변경하지 않고 대화 상자를 종료하려면 esc 키를 누르십시오.

글꼴 크기 및 테이블 테두리 크기 변경

글꼴 크기를 변경하려면 글꼴 메뉴로 이동해 글꼴 크기를 선택하십시오. 편집기는 메뉴에 크기 조정 대화 상자를 표시하고 대화 상자에 초점를 맞춥니다.

테두리 크기를 변경하려면 테이블 테두리 크기 도구 모음 항목으로 이동한 후 테이블 테두리 크기를 선택하십시오. 편집기는 메뉴에 크기 조정 대화 상자를 표시하고 대화 상자에 초점를 맞춥니다. 참고: 테이블 테두리 크기 도구 모음 항목은 커서가 테이블 안에 있을 때에만 사용할 수 있습니다.

크기 조정 대화 상자 내에서 tab 키를 누르면 다음 컨트롤로 이동합니다. 이전 컨트롤로 이동하려면 Shift+tab을 누르십시오.

크기를 수정하려면 크기 입력 상자에 새 값을 입력하십시오. 예: 14px 또는 1em. 변경 내용을 실행하려면 Enter 키를 누르십시오. Enter 키를 누르면 대화 상자가 닫히고 문서에 초점이 맞춰집니다.

대화 상자를 종료하지 않고 크기 증가나 감소 단추를 활성화하여 크기를 변경할 수 있습니다. 증가 또는 감소 단추로 크기를 변경하면 현재 단위 값을 유지하면서 선택한 요소의 크기를 즉시 변경합니다. 크기 대화 상자를 종료하려면 esc를 누르십시오.
