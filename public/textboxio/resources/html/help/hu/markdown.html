Markdown formázás

A markdown egy HTML-struktúrák létrehozására és formázásra használható szintaxis, mely billentyűparancsok használata vagy menük elérése nélkül működik. A markdown formázás használatához adja meg a kívánt szintaxist, és utána üsse le az enter vagy space billentyűt.
