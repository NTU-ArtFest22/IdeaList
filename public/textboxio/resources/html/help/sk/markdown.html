Formátovanie Markdown

Markdown je syntax na tvorbu HTML štruktúr a formátovania bez potreby používať klávesové skratky či vstupovať do menu. Na použitie formátovania Markdown zadajte požadovanú syntax a stlačte tlačidlo Enter alebo medzerník.

Syntax formátovania pomocou klávesnice
Syntax Výsledok v HTML 
# Najväčší nadpis
 
<h1> Najväčší nadpis</h1>
 
## Väčší nadpis
 
<h2>Väčší nadpis</h2>
 
### Veľký nadpis
 
<h3>Veľký nadpis</h3>
 
####  Nadpis
 
<h4>Nadpis</h4>
 
##### Malý nadpis
 
<h5>Malý nadpis</h5>
 
###### Najmenší nadpis
 
<h6>Najmenší nadpis</h6>
