Instruções de acessibilidade
Ativando a navegação pelo teclado

Para habilitar a navegação pelo teclado na barra de ferramentas, pressione F10 para Windows ou ALT + F10 no Mac OSX. O primeiro item na barra de ferramentas será realçado com um contorno azul, indicando que está selecionado.

Movendo-se entre grupos

Os botões na barra de ferramentas com funções assemelhadas são separados em grupos. Quando a navegação pelo teclado estiver ativa, pressione a tecla Tab para mover a seleção para o próximo grupo e Shift+Tab para mover a seleção para o grupo anterior. Ao pressionar Tab no último grupo, você retorna ao primeiro grupo de botões.

Movendo-se entre itens ou botões

Para voltar e avançar entre os itens, use as teclas de seta. Os itens serão selecionados em sequência dentro do próprio grupo. Para ir para o grupo seguinte, pressione Tab e use as teclas de seta.

Executando comandos

Para executar um comando, leve a seleção até o botão desejado e pressione espaço ou Enter.

Abrindo, navegando e fechando menus

Quando um botão de barra de ferramentas contiver um menu, pressione a barra de espaços ou Enter para abrir o menu. Quando o menu abrir, o primeiro item estará selecionado. Use as teclas de seta para navegar pelo menu. Para subir ou descer pelo menu, pressione as teclas de seta para cima e para baixo, respectivamente. Isso também se aplica aos submenus.

Os itens de menu que possuem submenus são identificados por um símbolo de divisa. Use a tecla de seta que corresponde à direção da divisa para expandir o submenu, e a tecla de seta oposta para fechá-lo.

Para fechar qualquer menu ativo, pressione a tecla Esc. Quando um menu é fechado, a seleção volta à posição anterior.

Modificando e removendo hiperlinks

Para editar ou remover um link, navegue até o menu Inserir e selecione Inserir Link. O editor exibe a caixa de diálogo de edição de link.

Edite o link inserindo o novo url na caixa de entrada de atualização de link e pressionando Enter. Remova o link do documento selecionando o botão de remover. Para sair da caixa de diálogo sem fazer alterações, pressione Esc.

Alterando o tamanho da fonte e da borda da tabela

Altere o tamanho da fonte navegando até o menu de fontes e selecionando o tamanho. O editor exibe a caixa de diálogo de tamanho no menu e define o foco nela.

Altere o tamanho da borda da tabela navegando até o item relacionado da barra de ferramentas e selecionando o tamanho da borda. O editor exibe a caixa de diálogo de tamanho no menu e define o foco nela. Nota: o item da barra de ferramentas para dimensionar a borda da tabela só estará disponível quando o cursor estiver dentro de uma tabela.

Na caixa de diálogo de tamanho, pressione a tecla Tab para mover a seleção para o controle seguinte. Pressione Shift+Tab para mover a seleção para o controle anterior.

Modifique o tamanho inserindo o novo valor na caixa de entrada de tamanho. Por exemplo, 14px ou 1em. Para enviar alterações, pressione Enter. Observe que, ao pressionar Enter, você fecha a caixa de diálogo e volta o foco ao documento.

Altere o tamanho sem sair da caixa de diálogo ativando os botões de aumentar tamanho ou diminuir tamanho. A alteração de tamanho com os botões de aumentar ou diminuir modificará imediatamente o tamanho do elemento selecionado, enquanto o valor de unidade em uso é mantido. Saia da caixa de diálogo pressionando Esc.
