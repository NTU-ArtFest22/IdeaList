Formatação com Markdown

Markdown é uma sintaxe para a criação de estruturas HTML e formatação sem a necessidade de usar atalhos de teclado ou menus de acesso. Para usar a formatação Markdown, insira a sintaxe desejada, seguida da tecla Enter ou Barra de Espaços.

Sintaxe de Formatação de Teclado
Sintaxe Resultado em HTML 
# Maior Título
 
<h1> Maior Título</h1>
 
## Título Maior
 
<h2>Título Maior</h2>
 
### Título Grande
 
<h3>Título Grande</h3>
 
####  Título
 
<h4>Título</h4>
 
##### Título Pequeno
 
<h5>Título Pequeno</h5>

<h6>Menor Título</h6>
