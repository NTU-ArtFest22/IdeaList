協助工具說明
啟動鍵盤導航

若要啟用工具列上的鍵盤導航，如 Windows 請按 F10，或 Mac OSX 請按 ALT + F10。工具列上的第一個項目將以藍色邊框反白，代表已選擇狀態。

在群組間移動

工具列內的按鈕均由類似動作的群組分隔。啟動鍵盤導航時，按下 tab 鍵將移動選項至下一個群組，而按下 shift + tab 則會將選項移回先前的群組。在最後一個群組上按下 tab 將切換回按鈕的第一個群組。

在項目或按鈕間移動

如要在項目間來回移動，請使用方向鍵。項目將在其群組內切換，如要跳至下一個群組，請按下 tab 鍵並使用方向鍵周遊群組。

執行命令

若要執行命令，請導覽選項至所需的按鈕並按下空格鍵或 enter。

開啟、導覽及關閉功能表

當工具列按鈕包含功能表時，按下空格鍵或 enter 將開啟功能表。功能表開啟時，將選擇第一個項目，請使用方向鍵導覽功能表。如要上下移動功能表，請分別按下向上或向下方向鍵，移動子功能表的方式亦相同。

含有子功能表的功能表項目會有 ＞形符號表示。使用對應 ＞形符號方向的方向鍵將展開子功能表，而反方向的方向鍵將將關閉子功能表。

如要關閉任何啟動的功能表，請按下 Escape 鍵。關閉功能表後，選項將還原回上一個選項。

編輯或移除超鏈接

如要編輯或移除鏈接，請導覽至插入功能表，然後選擇插入鏈接。編輯器顯示編輯鏈接對話方塊。

在更新鏈接輸入方塊中輸入新網址並按下 enter 即可編輯鏈接。選擇移除按鈕以移除文件中的鏈接。如要結束對話方塊且不變更，請按 esc。

變更字體大小及表格框線大小

導覽至字體功能表並選擇字體大小以變更字體大小。編輯器在功能表中顯示大小對話方塊，並設定對話方塊的焦點。

導覽至表格框線大小工具列項目並選擇表格框線大小，以變更框線大小。編輯器在功能表中顯示大小對話方塊，並設定對話方塊的焦點。注意：表格框線大小工具列項目僅適用於游標在表格內時。

在大小對話方塊中，按下 tab 鍵可移動選項至下一個控制項。按下 shift+tab 可移動選項至上一個控制項。

在大小輸入方塊中輸入新數值，即可修改大小。例如，14px 或 1em。如要提交變更，請按 enter。請注意，按下 enter 將關閉對話方塊，且畫面將返回文件。

透過啟動增加大小或減少大小按鈕，以變更大小且不結束對話方塊。利用增加或減少按鈕變更大小，將立即變更所選元素的大小，同時保留目前的單位值。按 esc 結束大小對話方塊。
