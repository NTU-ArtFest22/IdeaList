Formato Markdown

Markdown es una sintaxis para crear estructuras y formatos en HTML sin tener que utilizar atajos del teclado ni menús de acceso. Para utilizar el formato Markdown, introduzca la sintaxis deseada y a continuación pulse la tecla intro o la barra espaciadora.
