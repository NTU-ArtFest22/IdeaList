Formattazione Markdown

Il markdown è una sintassi per la creazione di formattazione e strutture HTML senza bisogno di usare le scorciatoie da tastiera o i menu d’accesso. Per utilizzare la formattazione markdown, inserire la sintassi desiderata e premere il tasto invio o spazio.

Sintassi di formattazione da tastiera
